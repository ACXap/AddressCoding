Столбец с адресами, без заголовка
Кодировка входных данных должна быть UTF-8, выходных будет UTF-8

1) Кнопка «Выбрать файл с данными» (можно просто перетащить файл на окно программы)
2) Имя файла для сохранения заполнится по шаблону: «Дата_Имя файла_Количество записей в файле»
3) Если нужно поменять имя файла для сохранения, кнопка: «Выбрать файл для сохранения»
4) Если нужно итоговый файл разбить на части (по строкам), ставим галочку и указываем нужное количество строк (итоговое число строк в файле = количество строк + строка заголовок)
Если итоговые данные разбиты по файлам, имена файлов: «Имя файла_Номер части»
5) Жмем кнопку: «Орпонизировать». Ждем
6) Жмем кнопку: «Сохранить данные»
7) В настройках нужно настроить подключение к сервису орпон и проверить это подключение
